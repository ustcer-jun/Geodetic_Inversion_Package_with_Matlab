# Geodetic Inversion Package with MATLAB
Package used to do kinematic slip inversion using geodetic data.

Ref: 
For use of code: Xu, X., Tong, X., Sandwell, D. T., Milliner, C. W., Dolan, J. F., Hollingsworth, J., ... & Ayoub, F. (2016). Refining the shallow slip deficit. Geophysical Journal International, 204(3), 1867-1886.

For phase-gradient constraints: Zhang, Y., & Xu, X. (2026). Constraining shallow slip deficit with phase gradient data. Geophysical Journal International, 244(1), ggaf427.
